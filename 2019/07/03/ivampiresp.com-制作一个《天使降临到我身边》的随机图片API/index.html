制作一个《天使降临到我身边》的随机图片API 由 ivampiresp.com

 内容可能已经失效
本文章对应的实际内容可能已经发生改变，部分链接可能失效。

俗话说的好：“百合三连，彻夜难眠。”
《天使降临到我身边》这部百合番我并没有看完，就到wikipedia上找到了官网http://watatentv.com 然后就开始抓图片了http://watatentv.com/story01.html
每集6张，一共70张。
预览图：

难度：十分简单
这次我并不用之前一个一个复制文件的方法，还是从Sakura主题cover目录中提取了1个文件，就用它来做随机图片吧。
这次不用找图床，直接找个免费虚拟主机(如果你主机带宽够大可以搭建到你自己服务器上)，我用的是https://infinityfree.net/ 建议用它的子域名，然后再建一个，最后搭建API。
把ftp工具准备好，然后到后台获取ftp用户名密码还有服务器。

紧接着下载上传并解压这个文件到新建子域名目录的htdocs下

https://happy-friends.ga/2019/07/Sakura-cover.zip

我上传到了子目录，因为主目录我要做两个API之间自动切换，代码我会放下来。

最后把下载到的图片放gallery文件夹即可。

图片我已经打包好了，下载链接：

http://ivampiresp.loveslife.biz/download/天使降临到我身边.zip

如果你还要对接其他图片API的话可以试试这个代码，

<?php
$urls = [
  'https://ivampiresp.com/wp-content/random.php',
  'https://api.lo-li.art/tianshijianglindaowoshenbian/index.php',
];
header('Location: '.$urls[mt_rand(0, count($urls)-1)]);

把上面代码保存为一个php文件即可，上面内容改成你的或者用我的。
这样就可以随机显示两个随机图片API中的随机图片啦。
