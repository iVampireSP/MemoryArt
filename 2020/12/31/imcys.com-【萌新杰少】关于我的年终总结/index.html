【萌新杰少】关于我的年终总结 由 imcys.com

前言

一转眼就又过去了一年呢，本站的创建时间也即将到1年了，这一年里我的收获也不少，在2020年我们也经历了严峻的考验——新冠疫情。

回望一年，不是我说呀，今年确实过的太快了，不知道是不是因为疫情影响，现在马上要春节了，感觉上次过就在前几天一样，啊这，太恐怖了。

一年的收获

这个，谈起收获，就不得不说今年一年的经历啦，今年因为疫情，我的高一下学期生活飞快的过去了，我也在这半年收获了许多的朋友，只是现在分班了，很可惜，其实也舍不得出来。但是，我以后想学计算机方面的专业，和能有更多可能考上大学，我不得不选择理科，走出自己最早的班级。

下半年，我认识了许多新的同学，也结识了许多新朋友，新的班级我也很快的适应了，这也是我开始最担心的事情，现在看来是多虑了。能结识到新的朋友，这也是一件不错的事情，只是现在要上晚自习了，嗯，到了理科班，学习emmm其实我感觉自己学的很菜，但这次排名大概就是在班上中上，最好的一次考过班上第8呢。

这一年，我也进行了学业水平考试，也就是会考，计算机是A，剩下的理化这些目前还不知道，现在还是比较紧张的。

除过这些收获呢，我在B站也有了更多的粉丝，而且自己的PHP和Android技术都有大的提升，这可以从今年发布的项目看到，下面我就要提及这些项目。

创建的项目

2020年，随着不断对PHP的了解，我也做了一些配合APP和QQ机器人运行的工具，那么现在来回顾一下我们制作了WEB端的哪些项目？

萌字号

https://mzh.mxjs.xyz/html/cx.php

emmm首先来说一下萌字号吧，嘶，虽然这个他，，，，，

萌字号，是一款在WEB端和APP端运行的群员违规记录备案系统，我们可以利用萌字号来为自己的群进行信息的备份，目前正在我自己的粉丝群使用。

这个也是我第一个做的较为完整的一套项目，尽管它有非常多的不足，但这也是我接触PHP后第一个处女作程序，哈哈，现在看起来好菜啊。

御坂妹妹API

御坂妹妹API，是我目前较为骄傲的一个项目，因为目前有不少人在用。这个项目是我今年后半年搞的一个项目，其实感觉自己学的PHP现在就一直用到了程序后端上面，写了不少爬虫，哈哈，但是也挺有成就感的。

目前这个项目正在前往 Misaka API上，emmm这个项目我等一下会介绍到它，御坂妹妹目前也是免费开放，供大家调用和使用，能帮上其他人我也很激动。

Misaka API

https://api.misakaloli.com

Misaka API是我后半年的作品，是一套准备写完整的API管理系统程序，也是MDUI的框架，后端这次是由我个人完全编写的，也涉及到了大量的数据库操作，并且支持了用户登录和调用统计，这个也是我目前PHP总水平的一个体现项目。

这个其实也是一个比较大的技术提升和实践锻炼，因为之前的项目都是零零散散，很少写这样一套的东西，所有觉得这个项目的意义很大。

另外也是因为它，我才加强了BILIBILI AS的WEB端API接口功能，写了一套B站的爬虫程序，目前也在给粉丝群的机器人使用，可以来体验哦。

剩下的WEB端项目，就是一些小的，比如TG电报机器人的PHP程序，或者发信程序。

WEB端的项目就介绍到这里了，下面我们来看看Android端发生了什么，按时间顺序来吧？

萌新云动漫

这个是今年IAPP写的程序，主要是可以免费无广告的观看番剧和电影，写这个的时候我还不会Java，所以就想把鸡肋的网页端写出个APP，然后让大家使用，写的比较菜，但这个程序也是有整体框架，写的也比较用心的。

但是这个APP现在停更了，我后面把源码发出来吧，现在也弃坑IAPP了，只要是用Java去写Android了。

发现酱

它可以识别同人图片，精确的找到图片原作来源，锁定作品链接，以及作者链接。

这个项目其实也是一个爬虫项目，对接其他接口完成的一个鉴别，给粉丝做视频的时候写的一个，我就不再多提啦。

BILIBILI AS

https://api.misakaloli.com/app/

BILIBILI AS，是今年后半年写的一个主打Android项目，也是我真正意义上的处女作，因为它是用Java写的，如今写文章的时候已经迭代到了1.0.7版本的APP，这在之前文章里提到过。

这个项目的伟大之处就在于它是Java写的Android项目，这也意味着我真正开始去做Android的项目程序，也希望大家能多多支持这个程序。

python

接下来就是python了，其实python是8月份玩的，会的不多，就是写写简单的爬虫项目。

这个大家可以自己看一下，我就不多介绍啦。
